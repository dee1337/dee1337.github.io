My Bare Minimum: Installs & Configs for Linux Mint 16 (Petra) - 64bit

Development:

Python-Ide (Spyder), Ruby, Universal-IDEs: Geany (lightweight), Eclipse(powerful + heavy))

{% highlight bash %} sudo apt-get install spyder geany geany-plugins eclipse ruby1.9.1-dev {% endhighlight %}

Useful Python Libraries and Tools:

{% highlight bash %} sudo pip install rope BeautifulSoup4 lxml virtualenv {% endhighlight %}

Prerequisites for Numpy & Matplotlib:

{% highlight bash %} sudo apt-get install python-dev libfreetype6-dev libpng-dev libjpeg8-dev libfreetype6-dev build-essential sudo apt-get install matplotlib {% endhighlight %}

SublimeText 3 install:

{% highlight bash %} sudo add-apt-repository ppa:webupd8team/sublime-text-3 sudo apt-get update sudo apt-get install sublime-text {% endhighlight %}

Graphic Design:

{% highlight bash %} sudo apt-get install krita {% endhighlight %}

system:

{% highlight bash %} sudo apt-get install htop {% endhighlight %}

settings and configs:

turn off bluetooth after booting up as default:

{% highlight bash linenos=table %} sudo su sudo gedit /etc/bluetooth/main.conf {% endhighlight %} >Look for an entry called 'Initially powered = true' and replace true with false.
