My Bare Minimum: Installs & Configs for Linux Mint 16 (Petra) - 64bit

Development:

Python-IDE (Spyder), Ruby, Universal-IDEs: Geany (lightweight), Eclipse(powerful + heavy)

installs Spyder (my fav. lightweight full-ide for python), Geany & Addons as alternative to gedit (Linux Mint only allows a very old version of gedit that lacks all the cool features modern versions have), Eclipse as heavy IDE for bigger projects with support of all programming languages (e.g. C++/C+/C, ObjC, Java, "Android", Python, ... ) {% highlight bash %} sudo apt-get install spyder geany geany-plugins eclipse ruby1.9.1-dev {% endhighlight %}

Useful Python Libraries and Tools:

installs rope for linting (very useful i.e. for spyder), BeautifulSoup + Lxml for WebScraping, virtualenv (you should know it) {% highlight bash %} sudo pip install rope BeautifulSoup4 lxml virtualenv {% endhighlight %}

Prerequisites for Numpy & Matplotlib:

installs the requirements that matplotlib needs (if "sudo apt-get install matplotlib" fails) before installing matplotlib itself: {% highlight bash %} sudo apt-get install python-dev libfreetype6-dev libpng-dev libjpeg8-dev libfreetype6-dev build-essential sudo apt-get install matplotlib {% endhighlight %}

SublimeText 3 install:

*adds the official SublimeText3 Beta to your repository & installs it: {% highlight bash %} sudo add-apt-repository ppa:webupd8team/sublime-text-3 sudo apt-get update sudo apt-get install sublime-text {% endhighlight %}

Office / PDF / Graphic Design:

Krita for (Vector-/+ Pen) Drawing

installs krita - a nice & intuitive drawing / vector drawing software with wacom tablet support {% highlight bash %} sudo apt-get install krita {% endhighlight %}

Adobe Acrobat Pro REAL Alternative: Master PDF Editor

finally, linux has a REAL and STABLE alternative to Adobe Acrobat Pro for editing the content of a pdf: Master PDF Editor is free for linux only and quite powerful (like it more than Acrobat Pro!)

Change Gimp to be more like Adobe Photoshop:

Change Gimp's (2.8 !!) UI to a Photoshop Style UI (BIG ui enhancement):
  • Download UI File from doctormo
  • extract the zip file into home folder (it's a hidden ".gimp-2.8" folder)
  • done :)

System:

{% highlight bash %} sudo apt-get install htop {% endhighlight %}

settings and configs:

turn off bluetooth after booting up as default:

{% highlight bash linenos=table %} sudo su sudo gedit /etc/bluetooth/main.conf {% endhighlight %} =>Look for an entry called 'Initially powered = true' and replace true with false.

Better Battery Management (Laptops) with TLP:

1. remove old management tool (if manually installed - basic Mint 16 does not have it): {% highlight bash %} sudo apt-get remove laptop-mode-tools {% endhighlight %} 2. add TLP repository, install and start TLP (only @ first time after install, you have to do it manually): {% highlight bash %} sudo add-apt-repository ppa:linrunner/tlp sudo apt-get update sudo apt-get install tlp tlp-rdw sudo tlp start {% endhighlight %}
