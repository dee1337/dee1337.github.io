IMPORTANT WINDOWS 7 STUFF:

sys-path invoke in terminal, e.g.: %systemroot%\Windows\System32\drivers\etc

REAL Windows 7 Space-Cleanup (fighting with a small SSD and Windows7-folder is already > 30GB ?):
- delete all files in "c:\Windows\Installer\$PatchCache$\Managed"
- delete all *.msp files in  c:\Windows\Installer\"
- forbid caching of installation updates: reg add HKLM\Software\Policies\Microsoft\Windows\Installer /v MaxPatchCacheSize /t REG_DWORD /d 0 /f

adding sth. to PATH (e.g. xampp): set PATH=%PATH%;C:\xampp\php 

installing python without vcversall.bat problems later on:

    
0.) USE STANDARD INSTALL PATHS!
1.) get python2.7 32bit & install
2.) get mingw and install it with base and c++ packages
3.) goto C:\Python27\Lib\distutils and create a distutils.cfg with the following content:
[build]
compiler=mingw32
4.) open a terminal (cmd or whatever) and put:
PATH = %PATH%;C:\MinGW\bin;C:\MinGW\msys\1.0\bin;
